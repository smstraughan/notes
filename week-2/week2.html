CSS Notes:

  • There are three ways to code CSS--inline, internal, and external.
    • Inline - CSS code is added in line to the html code. This is discouraged. Example in the code comments of this page.
    • Internal - CSS code is added to "style" section in the head of the html code. Still not prefered. Example commented in the comments of this page.
    • External - *Prefered* CSS code written in a separate .css document. Can be applied to multiple pages of a webpage easily. Then add code to document as a link in head. This is how this webpage is styled.
  • Examples of CSS properties are in the .css file that goes with this html doc
  • The CSS float property allows us to float content either to the left or right side and helps us to align it more
  • Pseudo classes allow us to style an element based on its current state
