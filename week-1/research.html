HTML Research Week 1



Prompts

  1. What are semantic tags in HTML and why are they important?
  2. HTML Validation: Explore the importance of HTML validation and how it can help ensure well-formed and error-free web pages. Provide a small contextual example.
  3. Explore the role of meta tags in HTML for SEO. Research common meta tags such as <title>, <meta name="description">, and <meta name="keywords">. Investigate their impact on search engine rankings and best practices for optimizing them.
  4. What is the difference between a block level element and an inline element?

Answers

  1. Semantic tags are tags that describe the content they explain
    • Improve readability across the internet
    • Improve accessibility for both mobile apps and responsiveness but also for screen reader users (inclusivity) maintains consistency throughout the web
    • Improves Search Engine Optimization (SEO) makes website contents more identifiable to search engines
    • Improves code readability for working on teams and makes coding in CSS easier because you can style by the semantic tags rather than by id attribute
  2. Validating HTML ensure error free webpages, making sure thing display correctly. It also helps to follow best practices and reccomendations of HTML specification. It helps improve accessibility of web pages for users with diabilities. Helps optimize performance of web pages by reducing the size, complexity, and loading time of HTML code
  3. Meta tags provide metadata about a webpage that help search engines understand page content and therefore influence how the page is displayed in search results, impacting page traffic and engagement
    • The title element is what appears as the clickable headline in search results
    • Meta description tags provid short summaries of the pages content (I think it is what pops up below the title click link)
    • Meta keywords seem to not be as important and might be spammy but used to influence SEO
  4. Block level elements start on a new line and take up the space available and inline elements take place in the line of text
    • Block elements help with layout and organization and ultimately with coding in CSS
