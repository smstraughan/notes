Inline vs Block

  • Inline and block are elements that help us structure and group content together
  • Block elements take up the entire width of their parent
    • Ex. Creating 2 block elements next two eachother will put them on different lines even if there is enough room for them side by side
  • Inline elements - multiple can fit on or in the same line
  • HTML structure elements fall in to block or Inline
  • Two main structure elements are div and span
    • Div is a block level element
    • span is inline - multiple can fit in same line
  • These elements help us organize our content that then we can refer to from CSS and JS
  • Divs and spans are practiced in this code but will not be visible on the rendered html
